Se creo el primer repositorio en la materia de herramientas computacionales, se hizo una introducción de git y se dejo un taller para enviar al profesor. Algunos comandos son

ls -a: Arhivos ocultos.
git init: Crea el repositorio.
git status: Estado del repositorio.
git add: Añadir archivos.
git commit -m: Comentarios hechos en un archivo.
git remote: conecta a un repositorio remoto.
git push: Envia los archivos del repositorio local al remoto.
git pull: Traer archivos.
